# Go RNN (Рекуррентная нейронная сеть)

Это проект, реализующий рекуррентную нейронную сеть (RNN) на языке Go. Цель проекта — продемонстрировать использование RNN для обработки последовательностей данных, таких как текст, и преобразование их в числовые представления с помощью нейросетевой модели.

Проект включает в себя основные компоненты, такие как:
- Реализация рекуррентной нейронной сети (RNN)
- Функции активации и их производных
- Преобразование текста в индексы слов с использованием словаря

## Описание

Этот проект реализует простую рекуррентную нейронную сеть, которая принимает текстовые данные, строит словарь, и преобразует текст в числовое представление. Проект включает следующие ключевые компоненты:
- **Структура RNN** для обработки входных данных и вычисления скрытых состояний.
- **Функции активации (например, tanh)** для обработки данных внутри сети.
- **Преобразование текста в индексы слов** с помощью словаря для подачи на вход нейронной сети.

## Установка

1. Убедитесь, что на вашем компьютере установлен [Go](https://golang.org/dl/).
2. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/your-username/go-rnn-neuro.git
   cd go-rnn-neuro
````

3. Установите зависимости (если они есть):

   ```bash
   go mod tidy
   ```

## Использование

1. После клонирования репозитория, откройте файл `main.go` для запуска программы.

2. Для запуска проекта выполните команду:

   ```bash
   go run main.go
   ```

3. Программа:

   * Строит словарь из текста.
   * Преобразует текст в индексы слов.
   * Проходит через последовательность с помощью RNN и выводит результаты.

Пример текста для обучения:

```go
texts := []string{
    "i am learning go",
    "go is awesome",
    "i love programming in go",
}
```

Программа построит словарь для слов из текста и затем выполнит преобразование текста в индексы, которые подаются на вход нейронной сети.

## Примеры

### Пример 1: Преобразование текста в индексы слов

```go
text := "i am learning go"
vocab := map[string]int{
    "i": 0,
    "am": 1,
    "learning": 2,
    "go": 3,
}

indices := TextToIndices(text, vocab)
fmt.Println(indices)  // Вывод: [0 1 2 3]
```

### Пример 2: Прямой проход через RNN

```go
inputText := "i am"
indices := TextToIndices(inputText, vocab)
hPrev := make([]float64, hiddenDim)

for _, index := range indices {
    x := make([]float64, inputDim)
    if index >= 0 {
        x[index] = 1.0
    }

    output, hNew := rnn.forward(x, hPrev)
    fmt.Printf("Output: %v\n", output)

    hPrev = hNew
}
```

## Важные функции

### 1. `TextToIndices(text string, vocab map[string]int) []int`

Эта функция преобразует текст в последовательность индексов слов. Она использует словарь `vocab`, где ключом является слово, а значением — индекс.

### 2. `BuildVocab(texts []string) map[string]int`

Функция строит словарь, который сопоставляет каждому уникальному слову в тексте уникальный индекс. Используется для преобразования текстовых данных в числовые.

### 3. `RNN.forward(x []float64, hPrev []float64) ([]float64, []float64)`

Реализует прямой проход через рекуррентную нейронную сеть для одного временного шага. На вход подается вектор `x` (one-hot представление текста), а на выходе — результат работы сети и новое скрытое состояние.
